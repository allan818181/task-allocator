# 🧩 Task Allocator

## 📘 Description
**Task Allocator** is a web-based system designed to simplify the process of assigning and managing tasks among team members or departments.  
It enables admins or managers to create, assign, track, and monitor tasks efficiently — improving productivity, accountability, and collaboration within an organization.

---

## ✨ Key Features
- 🔑 **User Authentication** – Secure login and registration for admins and users.  
- 🧩 **Task Creation & Assignment** – Admins can create and assign tasks to specific users.  
- 📅 **Deadline Tracking** – Each task includes a due date to manage workload effectively.  
- 📊 **Progress Monitoring** – Real-time updates on task status (Pending, In Progress, Completed).  
- 📨 **Notifications** – Users receive alerts when tasks are assigned or updated.  
- 📁 **Task Filtering & Search** – Quickly find tasks by name, user, or status.  

---

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Version Control:** Git & GitHub
